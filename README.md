# Zekai Luo Academic Homepage

This repository is prepared for GitHub Pages, following the simple academic
homepage style used by many personal research sites.

## Files for GitHub Pages

- `index.html`: homepage content
- `styles.css`: page styling
- `.nojekyll`: tells GitHub Pages to serve files directly

## Publish With GitHub Pages

Use a repository named:

```text
<your-github-username>.github.io
```

Then push these files to the repository's `main` branch. GitHub Pages will serve
the site at:

```text
https://<your-github-username>.github.io/
```

If you use a different repository name, enable GitHub Pages in repository
settings and choose the `main` branch with the repository root as the source.

## Local Preview

Open `index.html` directly in a browser, or run a small static server:

```bash
python3 -m http.server 8000
```
